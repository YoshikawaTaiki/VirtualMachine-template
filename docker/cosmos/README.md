# Cosmos

## Requirements(必要条件)

- [Docker](https://www.docker.com/)

1. `brew cask install docker`
2. starting docker.app

if you use Linux
 
- mysql-client
 
ex. `sudo apt install mysql-client`

## Usage

1. `make docker/build`
2. `make docker/start`
3. `$ gaiad version`
4. `$ gaiacli version`

If you can verify this version, this usage is over.
