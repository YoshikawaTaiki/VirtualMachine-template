# Cosmos

## Requirements(必要条件)

- [Docker](https://www.docker.com/)

1. `brew cask install docker`
2. starting docker.app

if you use Linux
 
- mysql-client
 
ex. `sudo apt install mysql-client`

## Usage

1. `docker build --rm -f "Dockerfile" -t cosmos:latest .`
